About

Understanding how things work — then building them better.

I'm Hassan — a Computer Engineering student at Lebanese International University, currently in my first year and already deep in the world of hardware, code, and everything in between. I got into engineering because I've always been the person in the room who needs to understand how things work. Not just use them — understand them. Case Study

Featured project.

Autonomous Firefighting Robot

Introduction to Engineering — LIU Final Project

A 4WD robot built with Arduino that autonomously detects fires using flame sensors and extinguishes them using an onboard fan module — no human input during operation. Designed and simulated in Tinkercad before physical assembly, then built and tested as a working prototype.

What I Built

  • A 4WD chassis controlled through an L298N driver for autonomous navigation
  • Flame-detection logic using analog sensors to locate a fire's direction
  • A fan-based extinguishing mechanism that triggers automatically on detection
  • A full circuit design simulated in Tinkercad before any physical build
  • Arduino UNO firmware handling sensing, decisions, and motor/fan output

What I Learned

  • Turning raw sensor input into real-time autonomous decisions
  • Motor driver wiring, PWM control, and H-bridge circuit behavior
  • The value of simulating first — catching design flaws before they're physical
  • Debugging systems where software and electronics intersect
  • Real-world sensors are noisier than any simulation lets on
